# Семестровая работа "Язык PL/SQL"

## Цель работы:
В первой части курса в рамках выполнения семестровой работы вам было выдано описание предметной области. В соответствии с ним была разработана структура данных.

Необходимо доработать структуру данных и разработать пакет процедур PL/SQL так, чтобы на его основе (добавив пользовательский интерфейс) можно было реализовать модуль "Управление безопасностью".

## Требования к модулю:
 - Создание нового пользователя, редактирование, удаление данных о нем.
 - Добавление (редактирование, удаление) роли.
 - Назначение роли пользователю, удаление роли пользователя.
 - Авторизация.
 - Проверка наличия у заданного пользователя требуемой роли.
 - Логирование авторизации пользователей. Указывается, какой пользователь, в какое время выполнил вход в систему.
 - Логирование действий в модуле – создание нового пользователя (редактирование, удаление), добавление (редактирование, удаление) роли, назначение (удаление) роли пользователю. Реализовать с использованием триггеров.
 - Логирование действий в модуле – авторизация, проверка роли. 
 - Сводный отчет по логам. Входные параметры – начальная дата, конечная дата, ФИО пользователя, действие. Все параметры необязательные (т.е могут быть указаны, а могут и нет).
